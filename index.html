About the Project

This is an open source project to help you design apps for Mac. You can use the kit in any way you like. Credit is not required. Please don’t distribute the kit yourself, but instead link people to this site: macos-uikit.xyz

Improve the UI Kit

Did you found anything incorrect or missing? Feel free to report problems or request new elements on GitHub. You can also contribute directly to the UI Kit by adding more elements.

Can I get it for free?

Yes, the UI Kit is free and open source, though I appreciate any support. It helps me to keep the UI Kit up to date. Alternatively, you can download it on GitHub.
